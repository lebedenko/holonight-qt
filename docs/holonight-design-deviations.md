# HoloNight Design Deviations

This file records intentional palette choices that do not meet the strictest automated contrast gates.

## HoloNight Light Selection Text

`HoloNightLight` uses documented `onPrimary=#FFFFFF` on `primary=#3E7BDB`. The pair is approximately 4.15:1, below WCAG AA's 4.5:1 normal text threshold but above the 3:1 UI component threshold. The palette keeps the documented white selection text for consistency with selected controls and highlights.
